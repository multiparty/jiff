if_else(trueVal, falseVal, op_idopt) → {SecretShare}

Simulate an oblivious If-else statement with a single return value. Should be called on a secret share of a bit: 0 representing false, and 1 representing true If this is a share of 1, a new sharing of the element represented by the first parameter is returned, otherwise, a new sharing of the second is returned.
Parameters:
Name Type Attributes Default Description
trueVal SecretShare | constant the value/share to return if this is a sharing of 1.
falseVal SecretShare | constant the value/share to return if this is a sharing of 0.
op_id string <optional>
 auto_gen() the operation id which is used to identify this operation. This id must be unique, and must be passed by all parties to the same instruction, to ensure that corresponding instructions across different parties are matched correctly.
