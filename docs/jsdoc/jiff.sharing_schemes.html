(static) shamir_share(secret, parties_list, threshold, Zp, random) → {object}

Default way of computing shares (can be overriden using hooks). Compute the shares of the secret (as many shares as parties) using shamir secret sharing: a polynomial of degree: ceil(parties/2) - 1 (honest majority).
Parameters:
Name Type Description
secret number the secret to share.
parties_list array array of party ids to share with.
threshold number the minimimum number of parties needed to reconstruct the secret, defaults to all the recievers. [optional]
Zp number the modulos.
random function a function that provides a random number between 0 and the provided number-1 inclusive
