(static) shamir_share(jiff, secret, parties_list, threshold, Zp) → {object}

Default way of computing shares (can be overridden using hooks). Compute the shares of the secret (as many shares as parties) using Shamir secret sharing: a polynomial of degree: ceil(parties/2) - 1 (honest majority).
Parameters:
Name Type Description
jiff jiff-instance the jiff instance
secret number the secret to share.
parties_list Array array of party ids to share with.
threshold number the min number of parties needed to reconstruct the secret, defaults to all the receivers.
Zp number the mod.
