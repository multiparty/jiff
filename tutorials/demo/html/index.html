JIFF Tutorial

The JIFF repo is available at https://github.com/multiparty/jiff

This tutorial is interactive and runs in the browser. If you would like to follow along you can either run it on your computer (if you have node.js installed), or use the links at the bottom to follow. If you use the links below, you will not be able to run most of the code interactively, as it requires access to a server.

To download and run the tutorial use these commands: 
git clone https://github.com/multiparty/jiff.git
cd jiff
git checkout secdev
npm run tutorial
